# Sadaqah Jar

Sadaqah Jar is a simple Expo React Native app for tracking charitable giving, capturing notes, and celebrating weekly progress.

## Getting started

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start the app:
   ```bash
   npm run start
   ```

## Project structure

- `App.tsx` — App entry and navigation setup.
- `src/db` — SQLite helpers and seed data.
- `src/screens` — UI screens.
- `src/components` — Reusable UI pieces.
- `src/theme` — Colors, spacing, typography.
